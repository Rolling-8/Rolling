## 기술스택

<div align="center">
<img src="https://github.com/Haze-S/blog-contents/assets/87344625/9b752839-f3d1-4e32-b054-0e22cb90f4ce" />
</div>

프로젝트 빌드 툴로는 `Vite`를 이용하고 CSS는 `Styled-Components`, 텍스트 에디터는 `Quill` 라이브러리를 사용했습니다. 협업 도구는 기존에 사용하던 `Discord`와 `Github` 만 사용하여 진행했고, Github만으로 이슈 관리와 칸반보드 등을 사용할 수 있어서 추가적인 툴은 사용하지 않았습니다.
